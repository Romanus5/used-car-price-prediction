# Прогноз цен на б/у автомобили с помощью линейных моделей

Этот проект демонстрирует применение классической линейной регрессии и линейных моделей с регуляризацией (Ridge и Lasso) для предсказания стоимости подержанных автомобилей на основе реального датасета с Kaggle.

## Использованные технологии
* Python 3
* Pandas, NumPy (предобработка данных)
* Scikit-Learn (машиностроение и метрики)
* Matplotlib (визуализация результатов)

## Метрики качества моделей
По итогам экспериментов были получены следующие результаты на тестовой выборке:

| Модель | MAE | R² Score |
| :--- | :--- | :--- |
| **Linear Regression (Baseline)** | 1.22 | 0.8490 |
| **Ridge Regression (L2)** | **1.21** | **0.8501** |
| **Lasso Regression (L1)** | 1.21 | 0.8490 |

**Вывод:** Модель Ridge (L2-регуляризация) показала наилучшую точность. Модель Lasso успешно занулила избыточные признаки (например, тип топлива Petrol и количество владельцев Owner), снизив мультиколлинеарность.

## Как запустить
1. Установите зависимости: `pip install pandas numpy scikit-learn matplotlib`
2. Запустите скрипт: `python train1.py`
